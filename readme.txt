Конвертация вычислительных схем биоинформатического пакета UGENE в программу на языке Python при помощи модулей Biopython.

В UGENE встроен модуль, позволяющий создавать вычислительные схемы, выполняющие считывание, анализ и запись данных в различных био форматах. Однако, возможности создания ограничены в первую очередь теми инструментами, которые встроены в UGENE. Поэтому, возникает необходимость в инструменте, который может конвертировать схемы из пакета UGENE в код на Biopython, который хорошо расширяем в дальнейшем. 

Результат: n-e количество сконвертированных вычислительных схем на BioPython.
